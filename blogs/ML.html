TF serving

  • tf serving makes model version management & model serving very easy

  • Also it can follow batch inference, runs req in batches, set a timeout parameter so it would process all req in that timeout in a single batch.

  • Allows better resource utilization.

Installing

docker pull tensorflow/serving

Setting up

docker run -t --rm -v <directory>:<docker_img_name(can be any)> -p <port(eg 8605)>:<port_to_exposed_as(usually same)> tensorflow/serving  
tensorflow_model_server --rest_api_port=<port(eg 8605)> --model_config_file=<path_of_models.configs.a>

Use Postman to test or go to localhost:8605/v1/models/\ (its always v1)
